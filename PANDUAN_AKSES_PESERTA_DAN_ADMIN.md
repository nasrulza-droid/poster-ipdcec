# Panduan Akses Peserta dan Admin

Dokumen ini adalah SOP singkat untuk akses website IPDCEC 2026, ditujukan untuk peserta dan panitia admin.

## 1. Link Utama

- Halaman peserta: https://lomba-poster.ipdcec.online
- Halaman admin: https://lomba-poster.ipdcec.online/admin.html
- API (status): https://api.lomba-poster.ipdcec.online/api/health

## 2. Alur Peserta

1. Buka halaman peserta.
2. Masuk ke bagian Akun Pendaftar.
3. Klik Daftar Akun, isi:
   - Nama lengkap
   - Email aktif
   - Password minimal 8 karakter
4. Setelah berhasil daftar, lanjut Login dengan email dan password yang sama.
5. Setelah login berhasil, form pendaftaran akan terbuka.
6. Isi data pendaftaran dan unggah seluruh dokumen wajib.
7. Centang persetujuan, lalu klik Kirim Pendaftaran.
8. Jika sukses, peserta diarahkan ke halaman konfirmasi.

## 3. Alur Admin

1. Buka halaman admin.
2. Login menggunakan akun admin yang aktif.
3. Setelah login berhasil, admin dapat:
   - Melihat daftar pendaftar
   - Mengubah status pendaftaran
   - Menghapus data (jika diperlukan)
   - Export data CSV
   - Melihat log aktivitas admin

Catatan keamanan admin:
- Kredensial admin dibagikan oleh koordinator resmi, jangan ditaruh di grup publik.
- Setelah login pertama, segera ganti password jika diperlukan.

## 5. Troubleshooting Cepat

- Tidak bisa login peserta:
  - Pastikan email dan password benar
  - Coba refresh halaman lalu login ulang
- Form tidak bisa dikirim:
  - Pastikan semua field wajib terisi
  - Pastikan semua dokumen wajib sudah diunggah
- Admin tidak bisa akses data:
  - Pastikan login admin berhasil
  - Cek API health pada link status API

## 6. Checklist Operasional Panitia

- Cek akses halaman peserta
- Cek akses halaman admin
- Cek status API
- Cek data pendaftaran masuk di dashboard admin
- Simpan backup rutin sesuai jadwal
